Manual do WMS LanLimp


> Seja bem-vindo!

Este é um manual desenvolvido para facilitar a análise e a manutenção dos recursos envolvidos no WMS LanLimp.

Neste manual você vai encontrar informações detalhando o funcionamento das manutenções existentes no WMS LanLimp. Entre essas manutenções estão: Parâmetros, cadastros e movimentações de entrada e saída dos produtos.

Com isto, você poderá identificar de forma mais fácil onde cada ação é realizada, e terá mais praticidade e agilidade na hora de evoluir ou dar manutenção no código fonte.
